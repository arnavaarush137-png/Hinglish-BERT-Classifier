# Dataset Description

## Project
Automated Hinglish BERT Classifier for Vishing and Smishing Phishing Campaigns

## Classes
1. Legitimate
2. Smishing
3. Vishing

## Languages
- Hinglish
- English
- Roman Hindi

## Goal
To train a BERT-based machine learning model capable of classifying incoming text messages into Legitimate, Smishing, or Vishing categories.
